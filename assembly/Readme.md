# Assembly Utility Library

Welcome to the Assembly Utility Library! This file is a collection of essential macros and procedures designed to make x86 assembly programming easier, more modular, and more powerful. Whether you are a beginner or an advanced developer, you can use and extend these utilities in your own projects.

## Purpose
This library is part of a larger project to collect, improve, and share high-quality assembly code. The goal is to help students, hobbyists, and professionals write better, reusable, and more readable assembly programs.

## Features
- **OUTDEC PROC / MACRO**: Print signed decimal numbers easily.
- **INPUTNUM PROC / MACRO**: Read signed numbers from user input.
- **GETNUM MACRO**: Read positive numbers (0-99) from user input.
- **PRINTSTR / PRINTSTR1 MACRO**: Print strings to the screen.
- **GETSTR1 MACRO**: Read a string from user input.
- **PRINT_MATRIX / PRINT_ARRAY MACRO**: Print arrays and matrices in a readable format.
- **SORT_ARRAY MACRO**: Sort an array in ascending order.
- **READ_ARRAY MACRO**: Read an array from user input.
- **READ_LED_NUM MACRO**: Output a number to a hardware port (e.g., LED display).
- **DELLAY MACRO**: Delay execution for a specified time.

## How to Use
- Copy the required macros or procedures into your own assembly project.
- Follow the usage comments above each section for correct integration.
- All code is compatible with MASM/TASM and EMU8086.

## Contribution & Sharing
This project is open for contributions! If you have useful macros, procedures, or improvements, feel free to submit them. The aim is to build a comprehensive, well-documented, and community-driven assembly resource.

## Why Use This Library?
- **Save time**: No need to rewrite common routines.
- **Learn best practices**: Each section is documented for clarity.
- **Grow the community**: Share your knowledge and help others learn assembly.
